GRAYSCALE THEME - IMAGES DIRECTORY
===================================

This directory should contain the following images for the Grayscale theme:

1. bg-masthead.jpg
   - Hero/masthead section background
   - Recommended size: 1920x1080px or larger
   - Should be a high-quality image suitable for full-screen display
   - Will have a dark gradient overlay applied

2. bg-signup.jpg
   - Signup section background
   - Recommended size: 1920x600px or larger
   - Should complement the overall grayscale aesthetic
   - Will have a dark gradient overlay applied

3. grayscale-theme.png
   - Theme preview thumbnail for admin panel
   - Recommended size: 400x300px
   - Should show a representative screenshot of the theme

NOTES:
- Images should be optimized for web (compressed but high quality)
- Use JPG format for photos, PNG for graphics with transparency
- Consider using grayscale or muted color images to match theme aesthetic
- You can download sample images from StartBootstrap's Grayscale theme or use your own
